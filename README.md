# docker-images

Docker images to help build and test [Neon](https://github.com/neondatabase/neon) product

Images provided:

- rust images (based on 'summerwind/actions-runner' docker images with rust, python3.9 and other tools installed)
  - `neondatabase/rust:1.58`, `neondatabase/rust:stable`
  - `neondatabase/rust:1.60`, `neondatabase/rust:edge`
